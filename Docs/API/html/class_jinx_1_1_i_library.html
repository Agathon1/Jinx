Detailed Description

ILibrary represents a single module of script code.

Multiple scripts can be compiled as part of a single library. This interface allows the creation of native functions that script code can execute via RegisterFunction() and properties through the use of the RegisterProperty() function. You can also get or set property values using the GetProperty() and SetProperty() functions.
